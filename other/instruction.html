Getting Started with VoidOS
Welcome to the future. Follow these steps to get up and running with Void OS in no time.
1. Download Void OS and get the latest Alpha version of Void OS.
⚠️ Currently available for testing purposes only.
Download (Click Here)
Download
2. Create Your Bootable DriveUse tools like Rufus (Windows) or BalenaEtcher (Mac/Linux) to flash the .iso file onto a USB stick.
3. Install the OS
The Nav Bar is responsive and can easily be updated to reflect your branding. To edit:
  • Plug in your USB stick
  • Reboot your system and enter BIOS/UEFI
  • Boot from the USB drive
  • Follow on-screen instructions
Void OS comes with AI-guided installation to help you configure everything — even if you're not tech-savvy.
4. Initial Configuration
  • Connect to WiFi
  • Sync your preferences
  • Choose your AI modules (minimalist mode or full power stack)
5. Activate the OS
Use the provided Void Key or proceed in Trial Mode.
Cloud sync and some advanced features require activation.
6. Explore the Interface
Void OS isn’t just pretty — it adapts.
  • Right-click gestures
  • Voice triggers
  • Command palette (Ctrl + Space)
  • AI sidekick (VoidLink) for instant answers
